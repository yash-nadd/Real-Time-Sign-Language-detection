About This Project

This web application detects and translates Indian Sign Language into text and speech.

Using advanced machine learning models and computer vision, this application identifies hand gestures in real-time and converts them into meaningful text and speech output. It supports both alphabet & numbers and word-based recognition.

The goal of this project is to bridge the communication gap for individuals using Indian Sign Language, making interaction easier and more accessible.
